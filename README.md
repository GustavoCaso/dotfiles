# Dotfiles

## What will be install

- Homebrew
- Rbenv (Ruby Version Manager)
- n (Node Version Manager)
- Kiex (Elixir Version Manager)
- rcm (dotfile managment)
- Zsh (Shell)
- Zplug
- Atom
- Spotify
- Google Chrome
- DropBox


## Installation

```shell
$ git clone git@github.com:GustavoCaso/dotfiles.git .dotfiles
$ cd .dotfiles
$ ./install.sh
```
